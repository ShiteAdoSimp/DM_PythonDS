# Project Overview
This project analyzes French presidential election data from 2022 using Python. The dataset comes from [data.gouv.fr](https://www.data.gouv.fr/fr/datasets/r/182268fc-2103-4bcb-a850-6cf90b02a9eb) and contains detailed voting results at the commune level for all 12 candidates in the first round (April 10, 2022).
# Installation
Install Python dependencies with:
``` bash
pip install -r requirements.txt
```

Requirements:
- pandas
- great_tables

# Contents
The notebook main.ipynb explores the election data through several questions:
1. General exploration
2. Comparison of departmental scores with national averages.
3. Cartography
